## 🚀 What is Neo Editor?

Neo Editor is a **professional SQL client** that rivals paid tools like TablePlus ($79) and DataGrip ($89/year), but is **completely free** with **zero setup required**.

## 🎯 Quick Start

1. **Download** the installer for your platform
2. **Install** and launch Neo Editor
3. **Select Database** - SQLite is pre-selected with sample data
4. **Run Your First Query**:
   ```sql
   SELECT * FROM users LIMIT 5;
   ```
5. **Explore Features**:
   - Click **Export** to save results
   - Press **Ctrl+K** to format SQL
   - Open **Schema** to browse tables
   - Try **Snippets** for templates
